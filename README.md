1. 请根据数据建立数学模型,给出每个大巴车速度函数.
2. 根据你的求解判断司机是否有超速行为。

求解过程（第一问）
---
分以下几步
1. 求路程累加和（30个数）.
2. 求平均速度（30个数）.
3. 将这30个数插值（三次样条）.
4. fft.
5. 求出`vMean(t)`的表达式（[0，1]上的连续函数）.
6. 求出`x(t)`的表达式.
7. 求出`v(t)`的表达式.
